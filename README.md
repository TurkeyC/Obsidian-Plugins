# Obsidian Plugins

自己修改的部分 Obsidian 插件合集。

## 插件列表

### [Age Encrypt](obsidian-age-encrypt/)

基于 [age](https://age-encryption.org/) 加密库的 Obsidian 内容加密插件。

- **透明解密**：设置一次主密码，GUI 中所有加密内容自动解密为普通 markdown
- **双粒度加密**：支持加密选中段落或加密整个文件
- **落盘加密**：文件在磁盘上始终以密文形式存储
- **命令行解密**：附带 CLI 工具批量解密

### [RSVP 速读增强](rsvp-reader/)

Obsidian RSVP 速读增强插件，帮助用户快速阅读笔记内容。
